# 💸 Spendly - Expense Tracker App

Spendly is a full-stack expense tracking application built to help users manage their daily expenses, track spending habits, and visualize financial data in a simple dashboard.

---

## 🚀 Tech Stack

### Frontend
- React.js
- Tailwind CSS
- Chart.js
- Axios

### Backend
- Java
- Spring Boot
- Spring Web
- Spring Data MongoDB

### Database
- MongoDB

---

## ✨ Features

- ➕ Add new expenses (amount, category, date)
- ✏️ Edit existing expenses
- ❌ Delete expenses
- 📊 View total spending
- 📅 Filter expenses by month
- 💰 Budget limit alerts (warning system)
- 📈 Spending visualization using charts
